## What to build

Extend the CLI with an **`ogb mr`** command group and the first end-to-end read path. Add merge request DTOs, API path helpers, and `IOgbApiClient` methods for **list** (and shared **get** used internally). Wire `ogb mr list` with optional `--status` filter, human table output, and `--json` array output. Reuse existing auth, host resolution, repo context (`-R` / git remote), identifier JSON parsing, and exit-code conventions from `ogb issue`.
