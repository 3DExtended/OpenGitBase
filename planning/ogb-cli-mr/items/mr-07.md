## What to build

Implement **`ogb mr ready <number>`** to publish a Draft merge request to **Open** via the publish endpoint. Analogous to `gh pr ready`.

## Acceptance criteria

- [ ] POST `…/merge-requests/{n}/publish`
- [ ] Output shows transition to Open (or current state if already open)
- [ ] Non-author / invalid state failures surface API error detail
- [ ] Unit tests for ready handler
