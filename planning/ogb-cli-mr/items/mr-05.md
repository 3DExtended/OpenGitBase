## What to build

Add **`IGitBranchResolver`** to detect the current git branch from the working directory. Implement **`ogb mr create`** with `--title` (required), `--body` / `--body-file`, `--head`, `--base`, and `--draft`. Default **`--head`** to current branch when omitted. Default **`--base`** to repository default branch when discoverable via API; otherwise require explicit `--base`. Success output: MR number and web URL.

## Acceptance criteria

- [ ] `ogb mr create --title "…" --head feature --base main` creates MR via POST
- [ ] Omitted `--head` uses current git branch; clear error outside a repo
- [ ] `--draft` sets `isDraft: true`
- [ ] API errors (no commits ahead, duplicate active MR, same ref) surfaced to stderr
- [ ] Unit tests for branch resolver and create payload
- [ ] `--json` on create emits created MR object
